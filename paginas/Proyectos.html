Lenguaje De Marcas Y Gestion De Base De Datos

En este curso, exploraremos dos pilares fundamentales en la era digital: el Lenguaje de Marcas y la Gestión de Bases de Datos. El Lenguaje de Marcas nos sumergirá en la creación de páginas web efectivas, cubriendo desde HTML y CSS hasta JavaScript para añadir interactividad. Por otro lado, la Gestión de Bases de Datos nos adentrará en el mundo de la organización y recuperación eficiente de información, abordando el modelado de datos, SQL y la administración de bases de datos. Estas habilidades son esenciales en carreras relacionadas con el desarrollo web, análisis de datos y gestión de proyectos tecnológicos, proporcionando una base sólida para enfrentar los desafíos del mundo digital.

Asignaturas

GESTION DE BASE DE DATOS

  • Modelado de datos bases de datos relacionales
  • Consulta y manipulación
  • Normalización
  • Administración y seguridad

En cuanto a la asignatura de Gestión de Bases de Datos, se abordará el diseño de esquemas de base de datos y la realización de consultas mediante SQL, así como los principios de la normalización de bases de datos. También se explorará la administración y seguridad e integridad de bases de datos.

LENGUAJE DE MARCAS

  • Creación de estructuras web con HTML
  • Edicion de XML
  • Interactividad con lenguajes de marcas
  • Transformacion XML

En la asignatura de Lenguaje de Marcas, los estudiantes aprenderán a crear la estructura básica de páginas web utilizando HTML, aplicar estilos visuales mediante CSS y aprender sobre xml y todas las formas de utilizarlo.
